valores nulos

en Javascript existe una diferencia entre variables undefined y variables null, la primera
es cuando se crea la variable pero no se ha inicializado entonces Javascrit le asigna como valor indefinido, la
segunda es cuando el programador quiere hacer enfasis que ahun no le asignado ningun valor null. 

            
                //variables infefinida
                let indefinida;
                //variable null
                let nulo=null
                console.log("variable indefinida "+indefinida);
                console.log("variable nullas "+nulo);
Tambien existe un tipo de datos NaN, que significa no es un numero 
            
                //valor NaN no es un numero
                let noEsUnNumero="hola"*3;
                console.log(noEsUnNumero);

Datos compuestos

Hasta el momento hemos utilizado datos primitivos: number, string, boolean, undefined, null.
Ahora vamos a trabajar los datos compuestos: object={}, array=[], funcion(), class={}, etc.

primero empezemos con las funciones, a las funciones se les puede pasar parametros , como en el siguiente ejemplo: 
            
                //definir funcion
                function saludar(nombre, edad){
                    console.log(`mi nombre es ${nombre} y tengo ${edad} años `)
                }

                //invocar a la funcion
                saludar("Rodney Zapata",47);
En el case que se invocar la funcion solamente sin pasarles parametros, entonces Javasript colocara los valores como indefinided, Tambien se puede colocar valores por defectos a partir del 2015 y es de la siguiente forma: 
            
                
                    //definir funcion con valores de parametros por defecto
                    function saludar(nombre="", edad=0){
                        console.log(`mi nombre es ${nombre} y tengo ${edad} años `)
                    }
existe otro tipo de funcion llamada anonima, donde es una funcion dinamica donde no se conoce el nombre de la funcion ni el valor de la variable.
Tambien primero hay que definirla para despues ser invocadas en caso contrario genera error, en cambio las funciones anterior declarada pueden ser invocadas en cualquier parte del codigo inclusive antes de ser definida.

            
                //funcion expresada o anonima
                const funcionExpresion=function(){
                    console.log("funcion anonima, no se conoce el nombre de la funcion ni el valor de las variables");
                }
        
                //invocamos la funcion
                funcionExpresion();

Arrays

los array se define entre corchetes [], y puedes contener elementos de distintos tipos como numeros, caracter, booleanos inclusive,
pueden contener a su ves otro array, como se muesta a continuacion: 
            
                //array
                const pruebaArray=[15,"hola",true,["A","B","C"]];
                console.log(pruebaArray);
        
                //vemos elementos del array
                console.log(pruebaArray[1]) //nos muestra el elemnto Hola
                console.log(pruebaArray[3][1]) //nos muestra el elemento dentro de otro vector osea la letra B
a partir del 2015 se agregó que para definir un array se debe hacer de la suiguiente forma: 
            
                //array a partir del 2015
                const pruebaArray2=Array.of(17,"wellcome");
                console.log(pruebaArray2);  
        
                //tambien se puede crear usando la POO
                const objArray= new Array(1,2,"hola");
                console.log(objArray);
        
                //tambien se puede definir un array de 100 elementos que se llene automaticamente con el valor predeterminado false
                const dArray= new Array(100).fill(false);
                console.log(dArray);    
        
                //array de cololres
                const colores= new Array("Amarillo","azul","rojo");
                console.log(colores);
                
                //agregamos el color verde
                colores.push("verde");
                console.log(colores);

                //quita el ultimo elemento del array, quedando el array como estaba inicialmente
                colores.pop();
                console.log(colores);

                //ciclo FOR dentro una funcion asincronica
                colores.forEach(function(el,index){
                 console.log(`

    un objeto contienes diferentes tipos de datos, numero, caracteres, inclusive arrays y hasta otros objetos .
    Hasta se puede que el campo llame una funcion.
